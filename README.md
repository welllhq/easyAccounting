## 一个自用的资产盘点工具  
### 主要功能
- 创建不同的账本以区分不同类型的资产
- 支持按时间\盘点周期记录资产变动
- 可视化资产统计一览

### 特色
- 使用pyside6绘制UI
- 已打包成exe文件
- 几乎完全使用Qwen3-coder编写、调试代码
- 人工优化只占10%
